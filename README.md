# 🎯 Backpack2 安全版 刷分專用機器人

專為 Backpack 交易所設計的安全版自動化刷分與統計機器人  
目標：高效刷分 ✅ 低風險 ✅ 減少風控 ✅

---

## ✨ 功能簡介

- 自動掛單 + 自動吃單
- 安全分控機制
- 自動止損機制
- 成交數據記錄
- 支援 USDC / SOL 兩邊交易
- 完整自訂參數（API、區間、網格、倉位等）
